Git 버전관리
 - 업데이트 되고, 변경된 파일들의 진행과정을 볼 수 있음
 - 혹여나 업데이트 된 내용을 취소 하고 싶을 때, 전 버전으로 다운그레이드가 용이함

Git commit
 - 수정한 내용에 대한 간략한 설명과 함께 누가 언제 무엇을 수정하고 만들었는지 사진 처럼 찍음
 - 일기처럼 기록하는 것

Git push
 - 로컬 레포에 있는 내용, 파일을 공유 가능한 원격 레포에 밀어 넣어 클라우드 환경에 저장하는 것

Git pull
 - 원격 레포에 있는 내용을 내가 사용 할 수 있게 로컬레포에 당겨와서 다운 받는 개념

Git clone
 - 다른사람의 코드나 다른피씨로 싹 복사해서 같은 환경이나 코드를 사용 할수 있게 함
